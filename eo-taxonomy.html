‚ö† Key User Limitations

Fragmentation ‚Äî Multiple overlapping platforms make discovery and comparison difficult
Steep learning curve ‚Äî Time to onboard is often underestimated
Opaque pricing ‚Äî Costs and service tiers are hard to compare across providers
Non-customisable workflows ‚Äî Processing pipelines often cannot be modified by users
Vendor lock-in ‚Äî Migrating code between platforms is costly
No code sharing ‚Äî Reuse of algorithms across platforms is not facilitated
Sustainability concerns ‚Äî Project-based platforms may disappear after funding ends
IP restrictions ‚Äî Commercial code cannot be published to external platforms without security assurances

‚ö° Technological Enablers

Cost NoR Portal ‚Äî Compare pricing offers across platforms in one place
Discovery GEOSS Yellow Pages ‚Äî Catalogue of data providers addressing discoverability
Interop OpenEO API ‚Äî Prevents vendor lock-in by abstracting across cloud back-ends
Data ARCO & Cloud Optimised GeoTIFF ‚Äî Maximise EO data usability in the cloud
Standards OGC APIs ‚Äî New-generation standards for serving geospatial data on the web
Catalogue STAC ‚Äî Standardised indexing of spatio-temporal asset collections
